    An application of Generalized Linear Model based on Poisson Distribution

    The goal of this project was to use a generalized linear model based on the Poisson distribution to try to explain the number of fissures in a turbine wheel, as a function of the number of hours the turbine runs and the number of turbines running for a given period of time.

    The activity was of great importance for the understanding and handling of a significant tool for statistical analysis. GLMs are used as a modeling alternative in occasions where the response variable of a regression produces errors that do not follow a normal distribution. The analysis also included the checking of theoretical assumptions such as: linearity, homoscedasticity, autocorrelation of errors, in addition to the diagnosis.
